## 技术栈

- Vue 3
- Vite
- Pinia
- Vue Router
- Axios
- Less

## 安装依赖
```
npm install
```

### 启动开发服务器（热重载）
```
npm run dev
```

### 为生产环境打包
```
npm run build
```

### 代码格式化
```
npm run lint
```
